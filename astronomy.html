Astronomy Tool

Astronomy tool is an application built within Java using JavaFX for the UI. This application is currently used at Hobart and William Smith Colleges in the intro to astronomy course. It allows users to specify a longitude and latitude, and the RA / DEC of a star and graph the altitude and azimuth of the star according to their position.
